# Contributing to Axonate

Thanks for helping! A few conventions:

- **Branch + PR** — `main` is protected; open a pull request, don't push directly.
- **Discuss first** — for features, open a [Discussion](https://github.com/Axonate/axonate/discussions)
  or a [roadmap issue](https://github.com/Axonate/roadmap/issues) before large work.
- **Keep it small** — focused PRs; describe what and why.
- **No secrets** — never commit `.env`, keys, or tokens (push protection will block them).
- **Tests/builds green** — run the relevant build/smoke before pushing.

See the [roadmap](https://github.com/Axonate/roadmap) for what's planned and known issues.
